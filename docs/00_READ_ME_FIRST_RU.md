# READ ME FIRST: как читать эти документы

Эта папка создана для Claude Code. Она содержит очищенную постановку задачи и validation cases.

## Что означает Mathematica/Wolfram в документах

Только текстовую нотацию и совместимый формат обмена. Реализация должна быть на Python.

Примеры вида:

```text
IBPInput = <| "Variables" -> {x1,x2}, ... |>
```

нужно понимать как **текст, который Python-парсер должен принять**, а не как требование писать код на Wolfram Language.

## Математические формулы

Формулы `J(n,m)`, `div(QF)`, `Q·∂G = HG` и т.п. задают математические отношения. Они не являются исполняемым кодом и не должны быть механически копированы в Python без архитектуры.

## Success policy

`Status = Success` допустим только когда все финальные интегранды локально конечны при `epsilon=0`, coefficients восстановлены как rational functions, IBP rows прошли surface checks, а reconstruction прошёл независимую проверку.
